Guardrail Documentation

Setup and usage guide for Guardrail for Jira.

Overview

Guardrail helps Jira administrators review access, surface potential risk, and create remediation tasks in Jira from one focused workflow.

Requirements

  • An Atlassian Cloud site with Jira
  • Jira administrator access
  • Guardrail installed from the Atlassian Marketplace

Getting started

  1. Install Guardrail into your Jira Cloud site.
  2. Open the Guardrail admin page in Jira administration.
  3. Choose the Jira project that should receive remediation tasks.
  4. Select the issue type to use for remediation, such as Task.
  5. Save your configuration.

Using Guardrail

  • Generate access review snapshots across Jira projects.
  • Review project access and identify items that need attention.
  • Create remediation tasks in Jira for follow-up work.

Support

If you need help, contact Guardrail support using the support information listed on the Atlassian Marketplace listing.
